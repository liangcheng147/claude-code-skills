## 4. 用户约束与迁移边界

正文/科学内容保护范围：正文、摘要、公式、表格数据、图注/表注、引用 key、标签、图片内容和科学结论。对此类内容只允许进行格式、LaTeX 命令、环境、结构、路径、图表位置和目标模板包装方式转换，不允许改写原始内容或改变含义。

行政性声明/补充信息处理范围：基金信息、致谢、作者贡献、数据可用性、伦理声明、利益冲突等，应先提取源文事实含义，再按照目标模板的必须格式或选定示例的用词、标签、顺序和结构改写。允许模板化表达，但必须保证意思一致，不能新增或猜测事实。

科学性附录/补充材料处理范围：包含方法、结果、公式、表格、图注、数据或科学结论的附录和补充材料，按正文/科学内容处理。只允许转换目标模板支持的结构、命令、环境、路径、链接和包装方式，不允许改写科学内容或结论。
